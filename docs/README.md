# Project Documentation

This directory contains high-level documentation and architectural guides for the Smart Med Tracker project.

## 📁 Contents
- `ARCHITECTURE.md`: Detailed breakdown of the system design and data flow.
- `API_DESIGN.md`: (Recommended) Documentation for API endpoints.
- `DATABASE_SCHEMA.md`: Visualization and explanation of the PostgreSQL models.

## 📝 Documenting New Features
When adding a new feature, please update the corresponding guide in this folder to ensure the team stays aligned on the vision.
