# Inventory Management Module

## 📝 Responsibilities
- Managing stock levels for specific pharmacies.
- Tracking expiration dates and batch numbers.
- Automated alerts for low inventory.

## 📊 Models
- `Inventory`: Linked to `Pharmacy` and `Medicine`. Includes quantity, price, and expiry.
- `StockMovement`: Tracking historical changes (Add/Remove).
