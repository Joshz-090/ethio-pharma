# Pharmacy Management Module

## 📝 Responsibilities
- Managing Pharmacy store profiles in Arba Minch.
- Geographical mapping for location-based search.
- Pharmacy verification workflow.

## 📊 Models
- `Pharmacy`: Name, Address, License Number, GPS Coordinates, Status (Pending/Approved).
- `PharmacyLocation`: Detailed spatial data for Arba Minch sectors.
