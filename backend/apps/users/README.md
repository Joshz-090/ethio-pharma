# User & Authentication Module

## 📝 Responsibilities
- User account creation and management.
- Multi-role support: **Patient**, **Pharmacist**, and **Admin**.
- JWT-based authentication.
- Localization settings (Language: English/Amharic).

## 📊 Models
- `User`: Extended Django AbstractUser (email as username).
- `UserProfile`: Role, Phone Number, Language Preference.
